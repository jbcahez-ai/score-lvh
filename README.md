# Score LVH

Application web statique de score en direct pour quiz/tournoi.

## Structure

- `index.html`: squelette de la page
- `css/styles.css`: styles globaux
- `js/app.js`: point d'entree et wiring des evenements
- `js/dom.js`: references des elements DOM
- `js/state.js`: etat central de l'application
- `js/storage.js`: persistance `localStorage`
- `js/setup.js`: logique de configuration des equipes
- `js/scoreboard.js`: rendu du classement et des cartes equipes
- `js/utils.js`: fonctions utilitaires

## Lancer

Ouvrir `index.html` dans un navigateur moderne.
